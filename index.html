How?

I’m by far not a designer. I like a simple approach for creating this webpage, something that once set up gets out of my way. This is why i chose the following tools and techniques:

  • the content itself is stored as plain markup files with John Gruber’s syntax
  • this content is transformed into html with Discount
  • during the transformation, CSS is merged in, which is responsible for the appearance

With this approach, whenever I want to create some new page, it boils down to writing something in markup, and rebuild everything (with make)

If you’re interested in this, you can also take a look into the github repository for these pages.
